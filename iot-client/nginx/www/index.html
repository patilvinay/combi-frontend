IoT Client API

Welcome to the IoT Client API. This API provides access to IoT device telemetry data.

Available Endpoints:

GET /api/devices

Returns a list of all registered devices and their telemetry data.

Required header: X-API-Key: your-api-key

GET /api/devices/{deviceId}

Returns telemetry data for a specific device.

Required header: X-API-Key: your-api-key

POST /api/register-device

Registers a new device.

Required header: X-API-Key: your-api-key

Request body: {"deviceId": "your-device-id"}
